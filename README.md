## Setup

1. Clone the repository:

   ```bash
   git clone git@github.com:solohuck/React-Portfolio.git
   
2. Navigate to the project directory:

   ```bash
   cd React-Portfolio
   
3. Install dependencies:

   ```bash
   npm install
   
4. Start the development server:

   ```bash
   npm start
   
5. Open your browser and go to:
   ```bash
   http://localhost:3000
